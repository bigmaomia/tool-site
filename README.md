# 🔧 工具小铺 - AllInOne Tools

> 纯前端在线工具集合，所有数据处理在浏览器本地完成，不上传服务器，安全可靠。

## 🌐 在线地址

**GitHub Pages:** https://bigmaomia.github.io/tool-site/

## 🎯 功能列表

### 🖼 图片工具
| 工具 | 功能说明 |
|------|---------|
| 📉 图片压缩 | 上传图片，调整质量参数进行压缩 |
| 🔄 格式转换 | JPG / PNG / WebP / GIF 互转 |
| 💧 图片加水印 | 上传图片，添加自定义文字水印 |

### 📄 文档工具
| 工具 | 功能说明 |
|------|---------|
| 📎 PDF 合并 | 选择多个 PDF 文件合并为一个 |
| ✂️ PDF 拆分 | 选择 PDF，按页码范围拆分 |
| 🖼 PDF 转图片 | 将 PDF 每一页转为图片预览 |
| 📦 PDF 压缩 | 通过图片压缩方式减小 PDF 体积 |

### 🔧 开发工具
| 工具 | 功能说明 |
|------|---------|
| { } JSON 格式化 | 格式化美化或压缩 JSON，带错误提示 |
| .\* 正则测试器 | 在线正则表达式测试与匹配高亮 |
| 🔐 Base64 编解码 | 文本/图片 Base64 互相转换 |
| 📋 文本差异对比 | 左右对比两段文本，高亮差异行 |
| 🆔 UUID 生成器 | 批量生成 UUID/GUID，一键复制 |
| ⏰ 时间戳工具 | 时间戳与可读时间互转，支持多格式 |

## 🏗 技术栈

- **纯前端 SPA** — 无需后端服务器
- **Hash 路由** — 单页应用，前端路由
- **Canvas API** — 图片压缩、转换、水印处理
- **pdf-lib** — PDF 合并与拆分
- **PDF.js** — PDF 预览与图片导出
- **原生 JavaScript** — 零框架依赖

## 🚀 本地运行

```bash
# 直接用浏览器打开
open index.html

# 或用 Python 启动本地服务器
python3 -m http.server 8080
# 访问 http://localhost:8080
```

## 📦 部署

本项目为纯静态站点，可一键部署到任意静态托管平台。

### GitHub Pages（已部署）

1. 推送代码到 GitHub 仓库
2. 仓库 Settings → Pages → 选择 main 分支或 gh-pages 分支
3. 自动部署，享免费 HTTPS 和全球 CDN

### 自定义域名

支持绑定自己的域名，GitHub Pages 自动提供免费 HTTPS。
